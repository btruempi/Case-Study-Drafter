📝 Editor – Quick Reference

Select: Click any element. A blue border with 8 handles appears.

Move: Drag the centre of the selection border (cursor = ✥). Red snap guides appear on edges & centres like PowerPoint.

Resize: Drag any of the 8 corner / edge handles. All text boxes and images are resizable.

Edit text: Click any text element — the cursor appears immediately so you can type. Highlight any words, then a small format bar pops up above your selection for bold, italic, colour, size, etc.

Move a text box: Drag the small ⠿ teal dot that appears in the top-right corner of the dashed selection border.

Bullet lists: Click inside the bullet area and type. Press Enter to start a new bullet. Press Backspace on an empty bullet line to remove it. Select any bullet text and use the format bar to change colour, size, bold, etc.

Format: Select an element → use the toolbar strip (B / I / U, alignment, colour) or the Typography & Box sections in the Properties panel.

Undo / Redo: Ctrl+Z / Ctrl+Shift+Z, or the ↩ / ↪ buttons.

Delete: Press Delete or Backspace (when not editing text), or click 🗑 in the panel.

Add text box: Click ＋ Text P1 or ＋ Text P2 in the toolbar.

Insert image: Click 🖼 Image P1 or 🖼 Image P2 in the toolbar to add a free-form image box. Then click the box to choose a file, or drag & drop an image file directly onto it.

Text wrap: Select any element → see the Text Wrap section in the Properties panel. Choose In Line, Square, Wrap Right, Wrap Left, Behind, or In Front — exactly like Microsoft Word's Layout Options.

Insert link: Highlight text in any text box, then click 🔗 Link in the toolbar (or press Ctrl+K) → type the URL and click Apply. Ctrl+click to follow a link while editing. To link an image, select it → Properties panel → Image Link.

Snap to grid: Hold Shift while dragging to snap to an 8 px grid, or click ⊞ Grid in the toolbar to keep grid-snapping always on.

Replace images: Click an image → Properties panel → Choose file, or drop directly onto the slot.

Export: Ctrl+S or the Export JSON button → drag the JSON file back to Claude to regenerate the PDF.
