# Case-Study-Drafter
A self-contained, browser-based editor for creating and customizing TankScan / AquaPhoenix case study documents. No software to install — everything runs in the browser.

Features

Click to select any element; drag to move; drag handles to resize
Edit all text directly on the page — no separate editing window
Bold, italic, underline, bullet lists, font size, color, alignment
Insert hyperlinks into text or attach a URL to any image
Add free-floating text boxes and images on either page
Wrap text around images (square wrap)
Undo / redo (Ctrl+Z / Ctrl+Shift+Z), copy, paste, duplicate
Export layout as JSON to preserve all edits; re-import by dragging the file back in
Snap-to-grid and alignment guides

How to use

Open the link above in any modern browser
Click any element to select it, double-click text to edit
Use the Properties panel on the left to adjust typography, position, colors, and links
Click Export JSON (or Ctrl+S) to save your layout
